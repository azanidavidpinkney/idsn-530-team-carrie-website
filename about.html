About this project (yes it was real!)

Site Mission/How It Came to Be

1 Problem: Being challenged to learn a lot in a very short time, in a field you’re totally unfamiliar with, well…  sometimes you just need a little extra help… and sometimes that help can be hard to find, especially in webdev where it might take finding someone to comb through every line of your code.

2 Solution: Everyone in class has googled different aspects of learning webdev at one point or another, and saved different tutorials and articles that they found useful. Compiling all of these resources, tested and used by real students, provides that extra help to supplement class.

3 Tagline: No Tutor, No Problem.

Project Description

This site grew from a homework assignment and was our first group project in class. We started with the idea for the site and created a site flow chart, style guide with colors & fonts, and a shared document to compile the content. We then developed the site as a team, with some specific requirements from class.
